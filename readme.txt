# Solar Gen Set Calculator

**Solar Gen Set Calculator** is a robust, standalone application that helps you accurately calculate solar power generation system requirements based on appliance energy consumption. With an intuitive graphical user interface, this tool dynamically sizes batteries, solar panels, inverters, and other key components to suit your needs.

## Key Features

- **Intuitive User Interface:**  
  Easily enter appliance details such as rated power, usage hours, and quantity through a user-friendly GUI.

- **Dynamic System Sizing:**  
  Automatically computes critical specifications including:
  - Battery capacity (Ah)
  - Inverter size (W)
  - Number of solar panels
  - MPPT/SCC controller ratings  
  …and more, ensuring your solar system is optimized for efficiency.

- **Graphical Visualization:**  
  Generates a detailed diagram illustrating your solar setup’s layout and interconnections, enhancing clarity in design and troubleshooting.

- **Data Export:**  
  Saves appliance load schedules and solar configuration details to CSV files, allowing you to keep a record of your calculations and system requirements.

- **Standalone Executable:**  
  Run the application directly without needing a Python installation or additional dependencies.

## How to Use

1. **Download the Application:**  
   Obtain the `Solar_Gen_Set_Calculator.exe` from the official distribution package.

2. **Launch the Application:**  
   Double-click the executable file to start the program—no installation or additional setup is required.

3. **Input Your Data:**  
   - **Appliance Information:** Enter details such as rated power (W), usage hours, and quantity.
   - **Solar Parameters:** Adjust key settings like system voltage, depth of discharge (DOD), and solar panel wattage using the provided controls.

4. **View and Save Results:**  
   - The application will instantly calculate the required solar system configuration.
   - A graphical layout and detailed component specifications will be displayed.
   - Results are automatically saved to a CSV file for future reference.
